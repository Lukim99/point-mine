# 곡괭이 수리 비용표

| 곡괭이 | 수리 비용 |
|---|---|
| 나무 곡괭이 | 8P로 내구도 최대 5 회복 |
| 돌 곡괭이 | 내구도 1당 돌 ×2 |
| 녹슨 철 곡괭이 | 내구도 1당 석탄 ×1 |
| 청동 곡괭이 | 내구도 1당 구리 광석 ×1 |
| 강철 곡괭이 | 내구도 1당 철 광석 ×1 |
| 황금 곡괭이 | 내구도 1당 석탄 ×2 |
| 티타늄 곡괭이 | 내구도 1당 은 광석 ×1 |
| 백금 곡괭이 | 내구도 1당 은 광석 ×2 |
| 흑요석 곡괭이 | 내구도 1당 비취석 ×1 |
| 강화 합금 곡괭이 | 내구도 1당 흑요석 ×1 |
| 루비 곡괭이 | 내구도 1당 자수정 ×1 |
| 사파이어 곡괭이 | 내구도 1당 자수정 ×1 |
| 오리하르콘 곡괭이 | 내구도 1당 자수정 ×1 |
| 아다만티움 곡괭이 | 내구도 1당 자수정 ×1 |
| 아스트랄 곡괭이 | 내구도 1당 에메랄드 ×1 |
| 마스터 곡괭이 | 내구도 1당 다이아몬드 ×1 |

나무 곡괭이는 손상량이 5보다 적으면 남은 손상만 복구되며 비용은 동일하게 8P입니다.
