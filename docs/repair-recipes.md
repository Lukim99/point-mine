# 곡괭이 수리 비용표

나무 곡괭이는 수리할 수 없습니다. 나머지 수리 재료의 판매가는 지하 1층 기준 채굴 1회 기대 수익보다 조금 높게 설정했습니다.

| 곡괭이 | 내구도 1당 수리 비용 |
|---|---|
| 나무 | 수리 불가 |
| 돌 | 돌 ×2 |
| 녹슨 철 | 구리 광석 ×1 |
| 청동 | 석탄 ×2 |
| 강철 | 구리 광석 ×2 |
| 황금 | 철 광석 ×1 |
| 티타늄 | 은 광석 ×1 + 석탄 ×1 + 돌 ×1 |
| 백금 | 은 광석 ×2 + 석탄 ×2 |
| 흑요석 | 흑요석 ×1 |
| 강화 합금 | 흑요석 ×1 + 은 광석 ×1 + 철 광석 ×1 |
| 루비 | 자수정 ×1 + 석탄 ×1 + 돌 ×1 |
| 사파이어 | 자수정 ×1 + 금 광석 ×1 + 돌 ×1 |
| 오리하르콘 | 자수정 ×1 + 철 광석 ×1 + 석탄 ×1 |
| 아다만티움 | 아쿠아마린 ×1 |
| 아스트랄 | 에메랄드 ×1 + 흑요석 ×1 + 석탄 ×1 |
| 마스터 | 다이아몬드 ×1 + 은 광석 ×1 |
